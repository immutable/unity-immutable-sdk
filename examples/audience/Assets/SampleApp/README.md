# Immutable Audience — Sample App

A demo of the `ImmutableAudience` SDK that exercises every public API.

## Requirements

- Unity 2021.3 LTS or later.
- An Immutable publishable key (`pk_imapik-test-…` for sandbox, `pk_imapik-…` for production).

## How to use

1. Open `Scenes/SampleApp.unity` and press Play.
2. Paste your publishable key and click **Init**.
3. Use the controls to exercise the SDK; the log pane records every call and response.
